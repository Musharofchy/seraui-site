### üîí PatternLock ‚Äì Real World Usage

1.  **Register Pattern**  
    Use `mode="set"` ‚Üí capture pattern with `onSetPattern` ‚Üí save (hashed) in your DataBase.
    
2.  **Unlock**  
    Use `mode="unlock"` ‚Üí pass saved pattern as `initialPattern` ‚Üí check via `onUnlock`.
    
3.  **Security**
    
    -   Hash pattern before storing (like a password).
        
    -   Add retry limits to prevent brute force.
        
4.  **UI/UX**
    
    -   Works with mouse & touch.
        
    -   Add ‚Äúreset pattern‚Äù flow if forgotten.
        
5.  **Enhance**
    
    -   Confirm twice when setting.
        
    -   Optionally increase grid size (e.g., 4√ó4) for stronger locks.
