# Installation

## ⚡ TL;DR — Get Started in 30 Seconds

```bash
npx seraui@latest init && npx seraui@latest add button
````

Then use it:

```tsx
import { Button } from '@/components/button'

export default function Page() {
  return <Button>Click me!</Button>
}
```

---

## 🛠️Prerequisites

* **Node.js 18+** and npm / yarn / pnpm
* A **React/Next.js project** (or create one: `npx create-next-app@latest`)
* **Tailwind CSS** installed
  *(We’ll help if not set up yet — see the [Tailwind guide](https://tailwindcss.com/docs/guides/nextjs))*

---

## 🚀 Which Installation Method?

* **Already using shadcn/ui?** → Use [Option 1](#-option-1--use-shadcnui-recommended)
* **Starting fresh / want a custom registry?** → Use [Option 2](#-option-2--use-seraui-cli-alternative)
* **Just want to test a component quickly?** → Jump to [Quick Test](#-quick-test)

---

## ✅ Option 1 — Use shadcn/ui (Recommended)

If you’re already using shadcn/ui, **you’re done** — Sera UI uses the same setup.

**New to shadcn/ui? Run:**

```bash
npx shadcn@latest init
```

Then add a Sera UI component:

```bash
npx shadcn@latest add https://seraui.com/registry/button.json
```

📘 [Full shadcn setup guide →](https://ui.shadcn.com/docs/installation)

---

## 🚀 Option 2 — Use SeraUI CLI (Alternative)

Use the CLI to scaffold and manage components from the Sera UI registry.

### 1️⃣ Initialize your project

```bash
npx seraui@latest init
```

Example output:

```shell
✔ SeraUI initialized successfully!

🌐 Registry URL: https://seraui.com/registry

📦 Installed dependencies:
  - clsx
  - tailwind-merge

📁 Created folders:
  - lib/
  - components/

✨ Ready to add components!
```

📁 **What got created:**

* `lib/utils.ts` — Helper utilities (e.g., `cn()` for merging classes)
* `components/` — Where your UI components live
* `tailwind.config.js` — Updated with Sera UI theme tokens

---

### 2️⃣ Browse available components

```bash
npx seraui@latest show
```

---

### 3️⃣ Add a component

```bash
npx seraui@latest add button
```

Output:

```shell
✔ Button added successfully!
```

---

### ⚙️ Advanced CLI Options

* **Initialize with a custom registry**

  ```bash
  npx seraui@latest init --url https://seraui.com/registry
  ```

* **Add a single component from a custom registry**

  ```bash
  npx seraui@latest add button --url https://different-registry.com/registry
  ```

* **Set a default registry URL**

  ```bash
  npx seraui@latest config --url https://seraui.com/registry
  ```

## 🛠️ Troubleshooting

**“Command not found”?**
Make sure you’re using `npx` (not `npm`).

**Components not styled?**
Check that Tailwind is running:
`npm run dev` and ensure your `tailwind.config.js` includes:

```js
content: [
  "./components/**/*.{js,ts,jsx,tsx}",
  "./app/**/*.{js,ts,jsx,tsx}"
]
```

**Still stuck?**

* Open an issue: [GitHub](https://github.com/sera-ui/sera-ui)
